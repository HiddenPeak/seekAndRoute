## 构建wheels注意事项

- 每一步操作前进行日志记录
- 具有相同依赖关系的编译环境保持在同一个虚拟环境
- 保持环境变量干净，考虑创建build.sh
- 编译完成后，需要使用单独创建的运行时环境进行测试
- 测试完成后，清除编译器缓存
